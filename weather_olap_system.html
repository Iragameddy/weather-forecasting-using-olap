About This Project
Documentation
🌩️ Weather Forecasting System Using OLAP
B.Tech AI & Data Science Mini Project 2025–26 Dept. of CS & IT
Abstract: This system integrates real-time weather data acquisition with OLAP-based multi-dimensional analysis. Using a star-schema data warehouse, weather observations across Tamil Nadu are transformed into analytical insights via roll-up, drill-down, slice, dice, and pivot operations. Machine learning modules provide short-term forecasting with 87–94% accuracy.
Problem Statement: Traditional weather portals lack multi-dimensional analytics capability. Historical weather patterns, aggregated across time, location, and climate zones, remain underutilized for decision-making in agriculture, disaster management, and urban planning.

Database Schema
-- FACT TABLE CREATE TABLE weather_facts ( fact_id SERIAL PRIMARY KEY, time_id INT REFERENCES dim_time, location_id INT REFERENCES dim_location, climate_id INT REFERENCES dim_climate, season_id INT REFERENCES dim_season, temperature_avg DECIMAL(5,2), temperature_max DECIMAL(5,2), rainfall_mm DECIMAL(6,2), humidity_pct INT, wind_speed_kmh DECIMAL(5,1), pressure_hpa INT, aqi_value INT, uv_index INT ); -- TIME DIMENSION CREATE TABLE dim_time ( time_id SERIAL PRIMARY KEY, date_actual DATE, day_of_week VARCHAR(10), month_num INT, month_name VARCHAR(15), quarter INT, year INT, is_weekend BOOLEAN ); -- LOCATION DIMENSION CREATE TABLE dim_location ( location_id SERIAL PRIMARY KEY, city VARCHAR(100), district VARCHAR(100), state VARCHAR(100), country VARCHAR(50), latitude DECIMAL(8,4), longitude DECIMAL(8,4), altitude_m INT, climate_zone VARCHAR(50) );
